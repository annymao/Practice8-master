這次的 lab 要做一個翻牌遊戲。

遊戲開始:
首先按下S鍵遊戲會開始(記得把輸入法改成英文)，球會開始動，時間會開始倒數。要按下 start 鍵之後才可以開始翻牌。

背景音樂:
一進入遊戲就會開始放背景音樂，按下1可以停止播放，按下2再繼續播放。背景音樂是要import一個 minim 的 library，利用裡面的class來載入音樂以及播放。

遊戲玩法:
找到兩個相同的照片，其中翻到浩全的照片會+40分，其餘+10分，如果翻錯的話-5分。翻對的話，照片會縮小到消失(利用 Ani 達到效果)，而翻錯的話會照片停留一下下然後翻回去(利用millis()來算時間 delay)。

遊戲結束:
利用一個變數 matchpair 來記錄現在一共找到多少pair了。如果在時間結束之前找到5對，就贏了。如果時間到了，遊戲結束。

時間倒數:
時間倒數同樣是利用 millis() 來算他的遊戲開始時間到現在共經過多少秒，再用30去減。
